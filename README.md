module : FA.TI.CI1.EEF.FIN

Ce dépôt contient mon implémentation progressive d’un Ray Tracer en Java, structurée autour de plusieurs jalons pédagogiques.
Le projet vise à développer un moteur de rendu simple capable de :

* Comparer des images (Jalon 0)
* Classes de bases pour le calcul vectoriel (Jalon 1)
* Lecture du fichier de description de scène (Jalon 2)
* Premières images (sphères uniquement) (Jalon 3)
* Calcul de la couleur d'un point (Jalon 4)
* Ajout des ombres et de l'illumination de Phong (Jalon 5)
* Finalisation et bonus (Jalon 6)
  * -- Ajout des plans et des triangles
  * -- Surfaces réfléchissantes

L’objectif pédagogique est d’apprendre :

* la programmation orientée objet en Java
* les tests unitaires (JUnit 5)
* la structuration d’un projet Maven
* la manipulation d’images (Java AWT)
* l’architecture d’un moteur de rendu simple
* l’organisation du code et du versionnement Git

Technologies utilisées :

* Java 25
* Maven
* JUnit 5.8.1
* IntelliJ IDEA
* Git / GitHub
